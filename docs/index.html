Module theme_utils

This module contains several utilities for selecting and managing themes.

It allows us to set a primary theme and an alternate theme, so we can toggle them by means of the toggle_theme function, which can be assigned to a key binding.

This module also allows the user to interactively select a theme among all the installed ones, and to cycle through all the installed themes via the next_theme and previous_theme functions.

Functions

toggle_theme () Toggles between primary and alternate themes in the current buffer.
get_themes () Returns a list of available themes.
set_primary_theme (name[, env]) Sets the name and/or the environment of the primary theme.
set_alternate_theme (name[, env]) Sets the name and/or the environment of the alternate theme.
select_theme () Shows a dialog that allows the user to change the current theme.
next_theme () It sets the next theme in the list of the currently installed themes.
previous_theme () It sets the previous theme in the list of the currently installed themes.

Functions

toggle_theme ()
Toggles between primary and alternate themes in the current buffer. If the current buffer contains a theme different from the primary and the alternate theme, then the primary theme will be set.
get_themes ()
Returns a list of available themes.
set_primary_theme (name[, env])
Sets the name and/or the environment of the primary theme.

Parameters:

  • name string Name of the primary theme
  • env table Environment specific to this theme. It may override the fields of theme_env. (optional)
set_alternate_theme (name[, env])
Sets the name and/or the environment of the alternate theme.

Parameters:

  • name string Name of the alternate theme
  • env table Environment specific to this theme. It may override the fields of theme_env. (optional)

next_theme ()
It sets the next theme in the list of the currently installed themes.
previous_theme ()

Tables

theme_env
Global theme environment.

This can be used to override the default settings of the current theme. It will be passed as second paremeter of TextAdept's view:set_theme method.

Primary and alternate themes may further override the settings in this table. set_primary_theme and set_alternate_theme for more details.
